## 4. Responsive Design

Ứng dụng sử dụng TailwindCSS với responsive breakpoints:

| Breakpoint | Kích thước | Layout |
|-----------|-----------|--------|
| Mobile | < 640px | 1 cột, menu hamburger |
| Tablet | 640-1024px | 2 cột product grid |
| Desktop | > 1024px | 3 cột product grid, sidebar filter |
